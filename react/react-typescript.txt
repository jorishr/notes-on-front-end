#################
REACT + TYPESCRIPT
#################

CREATE REACT APP

	npx create-react-app <my-app> --template typescript

MANUALLY ADD TO REACT APP

Install		npm install --save-dev typescript

Script		In package.json add script: "build": "tsc"

Configure		npx tsc --init

In tsconfig.json, tell the compiler where the source code is and where the output should go (rootDir in src and outDir in build).

FILE EXTENSIONS

.ts is the default file extension while .tsx is a special extension used for files which contain JSX.

REFERENCE: https://reactjs.org/docs/static-type-checking.html
