# Bootstrap 4 vs. 3
- Global font size is 16px versus 14px. You adjust using rem.
- Grid system: xs - sm - md - lg - xl
- Panels, thumbnails and wells are now all "cards" 
- Geen Glyphicons meer. Externe fonts/icons gebruiken.
